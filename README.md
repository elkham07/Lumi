# Lumi | Premium Movie Discovery Platform

Lumi is a modern, cinematic movie discovery website designed for film enthusiasts. It provides a sleek user interface to explore trending titles, search for specific movies, and view detailed information using the TMDB (The Movie Database) API.

##  Features

- **Cinematic UI/UX**: A darker-themed, premium interface with glassmorphism effects and smooth transitions.
- **Dynamic Movie Grid**: Fetches real-time popular and trending movies from TMDB.
- **Live Search**: Find any movie instantly with the integrated search functionality.
- **Movie Details**: Click on any movie card to see an overlay with the overview, release date, and rating.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Interactive Branding**: The unique "Explore" hero button provides a minimalist entry point to the collection.

##  Technologies Used

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3 (Vanilla)**: Custom design system with CSS variables, Flexbox/Grid, and cinematic animations.
- **JavaScript (ES6+)**: Handles API integrations, dynamic DOM manipulation, and UI interactivity.
- **TMDB API**: Powers the real-time movie data and search results.
